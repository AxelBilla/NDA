<h1 align="center">HOW TO USE</h1>
<p align="center">Every step of the process has been split into a script, to ensure ease of use no matter the hardware.<br>However, you may use <b>main.py</b> to expediate the process and automatically go through all steps back-to-back</p>
<br>

## `python [SCRIPT] [SHEET]`

    (script)      [SCRIPT]: Script to use.
    (argument)    [SHEET]: Path to the sheet to process.    
  
  > [!NOTE]
  > Every script is executed in the same way. Only one argument may be provided, any superfluous argument will be ignored.
